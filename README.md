## 🗂️ Projects

### 1. 🎓 RAG AI Teaching Assistant
> Ask questions about your video course — get answers with exact video titles and timestamps

A fully local RAG pipeline that transcribes video lectures using Whisper, embeds the chunks using Ollama (bge-m3), and answers student questions using LLaMA 3.2 — all without any internet or API keys.

**Tech:** Python, Whisper, Ollama, LLaMA 3.2, bge-m3, scikit-learn, FFmpeg  
📂 [`/RAG-Teaching-Assistant`](./RAG-Teaching-Assistant)

---

### 2. 🛡️ PhishGuard — Phishing Website Detection
> ML-based URL classifier that detects phishing websites with ~96% accuracy

Enter any URL and the app predicts whether it is safe or a phishing threat using a Logistic Regression model trained on thousands of labeled URLs.

**Tech:** Python, Flask, scikit-learn, NLTK, Tailwind CSS  
📂 [`/PhishGuard`](./PhishGuard)

---

### 3. 🎬 VidsnapAI
> AI-powered reel generator — turn images + a description into a narrated short video

Upload a set of images and a text description, and VidsnapAI automatically generates a voiceover using ElevenLabs TTS and stitches everything into a 1080×1920 MP4 reel using FFmpeg.

**Tech:** Python, Flask, ElevenLabs API, FFmpeg  
📂 [`/VidsnapAI`](./VidsnapAI)
